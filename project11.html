Crypto Clustering Using K-Means & PCA

Clustered over 1,200 cryptocurrencies using unsupervised learning and visualized insights in 3D with Plotly. Deployed using Amazon SageMaker.

This guided project simulated a client-facing scenario in which a Big Four consulting firm assists a large investment bank in exploring the cryptocurrency market for a new portfolio product. The client needed a meaningful classification of tradeable cryptocurrencies to guide future investment strategies. The solution used unsupervised machine learning—specifically K-Means clustering—on preprocessed crypto data, followed by visualization and interpretation of the clusters.

Process Overview

  • Fetched cryptocurrency metadata from the CryptoCompare API.
  • Preprocessed the data by cleaning null values, converting categorical data using dummy variables, and scaling numerical features.
  • Applied PCA (Principal Component Analysis) to reduce dimensionality to three key components for visualization.
  • Used the Elbow Method to determine optimal cluster count, then performed K-Means clustering.
  • Visualized the clustered data in 3D using Plotly Express and created additional scatter plots and summary tables.
  • Deployed and tested the notebook using AWS SageMaker.

Finding the Optimal Number of Clusters

The Elbow Curve helped determine the optimal number of clusters (k) by plotting the inertia values across multiple k values.

3D Visualization of Crypto Clusters

The PCA-reduced data was used to visualize clusters of cryptocurrencies in three dimensions. Each point represents a crypto asset, colored by cluster assignment.

Total Coins Mined vs. Supply

Conclusion

This project demonstrated the potential of unsupervised learning to organize and classify large-scale cryptocurrency data. Through dimensionality reduction and clustering, we uncovered natural groupings and traits among crypto assets—delivering insights useful for portfolio design and investment decisions. Using AWS SageMaker ensured scalability and production-ready modeling.
